# Honey Cloud Coffee ☕

## Proyecto realizado en C# y Windows Forms.
###Los patrones de diseño son soluciones reutilizables a problemas comunes en el desarrollo de software. Estos patrones no son implementaciones específicas, sino más bien plantillas que se pueden seguir para resolver problemas de diseño en diversas situaciones. En este proyecto se implementó cuatro patrones para poder realizar este proyecto las cuales fueron el patrón fábrica abstracta, decorador, estado y event driven, para poder solucionar nuestro sistema de una cafetería digital  llamado honey moon coffee.
